Desenvolvemos uma aplicação web-front-end para testar o desenvolvimento do usúario em formato de quiz. 
Nosso quiz tem exibição sequencial de perguntas, validação automática da resposta selecionada, cálculo e exibição da pontuação final.
Nesse projeto usamos HTML5, CSS3 E JavaScript.
